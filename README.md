## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/Chat-App.git
cd Chat-App/client
```
2. Install dependencies
```bash
npm install
```
4. Start the development server
```bash
npm run dev
```
5. Open your browser and navigate to
http://localhost:5173
